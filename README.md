## 과제 제출 전 체크 리스트

- [ ] 터미널에서 `java --version` 실행하여 Java 버전이 21인지 확인
- [ ] 아래 명령어를 실행하여 실행하여 모든 테스트가 통과하는지 확인
    - Mac/Linux 사용자 : `./gradlew clean test`
    - Windows 사용자 : `gradlew.bat clean test`
- [ ] `BUILD SUCCESSFUL` 메시지를 확인
